# Losha Premium Storefront 🛍️✨

A fully functional, modern and highly aesthetic e-commerce homepage designed for premium apparel brands. 

## Features
- **Zero-Dependency Architecture**: Built with lightweight Vanilla HTML5 & CSS3. No build tools required—open it directly in your browser.
- **Premium UI Aesthetics**: Glassmorphism navbar, soft gradient blends, and carefully curated typography (Playfair Display & Outfit).
- **Responsive Layouts**: Implemented mobile-first practices. Looks flawless on mobile, tablet, and desktop devices.
- **Product Modules**: Includes pre-built grid layouts, hover micro-animations, and interactive UI swatches.

## How to View
To view your new store, simply open the `index.html` file in any modern web browser.

## Deploying to GitHub Pages
Since this is an HTML/CSS product, it is extremely easy to host for free!
1. Commit all files and push this directory to a new GitHub repository.
2. Navigate to your repository **Settings > Pages**.
3. Under the `Build and deployment` source, choose **Deploy from a branch**.
4. Select the main branch and root folder, then click save.
5. GitHub will automatically host your store live on the internet!
